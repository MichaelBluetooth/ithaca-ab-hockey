# Ithaca A/B Hockey Website

## Prereqs

1. Install [LTS NodeJS](https://nodejs.org/en)
2. Install required dev tools
```
npm install -g @angular/cli
npm install -g angular-cli-ghpages
```

## Setup

1. Clone the repo
2. Run `npm install`
3. Run `ng serve`
4. Make changes!
5. Run `npm run build-and-deploy`
